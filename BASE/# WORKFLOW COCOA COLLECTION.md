# WORKFLOW: COCOA COLLECTION

1. Administradora programa acopio.

2. Encargado de acopio recibe programación.

3. Se coordina con grupo base.

4. Socio entrega cacao.

5. Se pesa cacao.

6. Se genera recibo.

7. Se registra entrega.

8. Entrega se asigna a lote.

9. Contadora revisa entrega.

10. Sistema calcula valor.

11. Sistema aplica descuentos.

12. Contadora genera pago.

13. Administradora aprueba.

14. Sistema registra comprobante.

15. Socio puede visualizar pago.

# WORKFLOW: INPUT DELIVERY

1. Proyecto adquiere insumo.

2. Insumo ingresa inventario.

3. Administradora aprueba entrega.

4. Encargado de proyecto entrega.

5. Sistema registra entrega.

6. Si financiado:
   genera deuda.

7. Cuando socio entrega cacao:
   sistema propone descuento.

8. Contadora confirma descuento.

9. Deuda queda actualizada.
